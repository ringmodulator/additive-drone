# additive-drone

Looping drone instrument with additive synthesis, made with SuperCollider. This repo contains 2 pieces made with this instrument:

- `additive-drone.scd`: Pretty, relaxing drone piece that can be scaled to any length
- `looky-here.scd`: Piece created for live performance at [Looky Here](https://lookyheregreenfield.com/) in Greenfield, MA, 2026-03-05

The repo also contains 2 utilities related to the Looky Here piece:

- `looky-here-tuner.scd`: Guitar tuning reference
- `loop-recorder.scd`: For separately recording guitar loop for Looky Here piece
